# Menu de Números – C++

Aplicação de console escrita em C++ que permite ao usuário gerenciar uma lista de números inteiros por meio de um menu textual interativo.

## Funcionalidades

- Exibir todos os números da lista
- Adicionar um novo número
- Calcular a média dos números
- Mostrar o menor número da lista
- Mostrar o maior número da lista
- Encerrar o programa

## Conceitos utilizados

- Estruturas de controle (`do-while`, `switch`)
- Container da STL (`std::list`)
- Laço `for` baseado em intervalo (range-based for)
- Entrada e saída padrão (`cin` e `cout`)
- Validação básica de estado (lista vazia)
- Conversão e manipulação de caracteres (`toupper`)
